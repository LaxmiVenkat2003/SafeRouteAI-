# SafeRoute AI: Women’s Safety Navigation App

## Tagline
AI-powered safe routes and instant SOS alerts.

## Problem
Women face safety risks while traveling, especially at night or in unfamiliar areas.

## Solution
SafeRoute AI provides safer navigation routes, real-time SOS alerts, and AI-powered recommendations for nearby police stations and hospitals.

## Impact
Empowers women with confidence and security, potentially saving lives.

## Tech Stack
- Frontend: React / Flutter
- Backend: Node.js + Express
- Database: Firebase Firestore
- AI Tools: GitHub Copilot, ChatGPT, Claude, Cursor
- Deployment: Vercel / Render / Railway
